\makeproblem{Large \(N\) approximation of binomial distribution}{pr:poissonStirling:1}{

In \S 11.1 \citep{ma1985statistical} it is stated that the binomial distribution

\begin{equation}\label{eqn:poissonStirling:20}
\rho(n) = \binom{N}{n} p^n (1 - p)^{N-n},
\end{equation}

has the large \(N\) approximation of a Poisson distribution

\begin{equation}\label{eqn:poissonStirling:40}
\rho(n) = \frac{(\alpha v)^n}{n!} e^{-\alpha v},
\end{equation}

where \(N/V = \alpha\) is the density and \(p = v/V\), the probability (of finding the particle in a volume \(v\) of a total volume \(V\) in this case).

Show this.
} % makeproblem

\makeanswer{pr:poissonStirling:1}{

This is another Stirling's approximation problem.  With \(p = v \alpha/N\), and working with log expansion of the \(N!\) and \((N-n)!\) terms of the binomial coefficient we have

\begin{dmath}\label{eqn:poissonStirling:60}
\ln \rho
\approx
\lr{ N + \inv{2} } \ln N - \cancel{N} - \lr{ N - n + \inv{2} } \ln (N - n) + (\cancel{N} - n) - \ln n! + n \ln p + (N - n) \ln (1 - p)
=
\lr{ N + \inv{2} } \ln N - \lr{ N - n + \inv{2} } \ln (N - n) - n - \ln n! + n \ln \frac{v \alpha}{N} + (N - n) \ln \lr{1 - \frac{v \alpha}{N}}
=
\lr{ N + \inv{2} - N + n - \inv{2} -n } \ln N - \lr{ N - n + \inv{2} } \ln \lr{1 - \frac{n}{N}} - n - \ln n! + n \ln v \alpha + (N - n) \ln \lr{1 - \frac{v \alpha}{N}}
\approx
\ln \frac{(v\alpha)^n}{n!} - n
- \lr{ N - n + \inv{2} } \lr{- \frac{n}{N} - \inv{2}\frac{n^2}{N^2} } + (N - n) \lr{- \frac{v \alpha}{N} - \inv{2} \frac{v^2 \alpha^2}{N^2}}
=
\ln \frac{(v\alpha)^n}{n!} - n
+ n \lr{ 1 - \frac{n}{N} + \inv{2N} } \lr{1 + \inv{2}\frac{n}{N} } - v\alpha \lr{1 - \frac{n}{N} } \lr{1 + \inv{2} \frac{v \alpha}{N}}
\approx
\ln \frac{(v\alpha)^n}{n!} - n
+ n - v\alpha
\end{dmath}

Here all the \(1/N\) terms have been dropped, and we are left with

\begin{equation}\label{eqn:poissonStirling:80}
\ln \rho \approx \ln \frac{(v\alpha)^n}{n!} - v\alpha,
\end{equation}

which is the logarithm of the Poisson distribution as desired.
} % makeanswer
